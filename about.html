AI情報ブログとは

AI情報ブログは、注目しているYouTubeチャンネルからキーワードを拾い上げ、AIによる再調査と要約を経て記事化する自動化ワークフローの実験場です。MVP段階では GitHub Pages + GitHub Actions のみで構成し、外部サービスへの依存を最小限に抑えています。

  • 監視対象は data/sources.json で管理
  • 朝（JST 09:00）と夜（JST 21:00）に自動的にドラフト生成フローを実行
  • 生成されたドラフトは posts/generated-drafts/ に保存し、人手で公開可否を判断

今後は自動生成プロセスへのフィードバックループを加え、より信頼できる技術ブログとして拡張していきます。
