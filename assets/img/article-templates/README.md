# Article Template Images

このディレクトリにはブログ記事向けの WEBP テンプレート画像を配置します。ファイル名は `assets/img/articles/index.json` の `key` に合わせて `<key>.webp` を作成してください（例: `ai-core-01.webp`）。

## 運用ルール
- 画像サイズは 16:9 (推奨: 1200x675)。
- 色数やファイルサイズは 400KB 以下を目安に最適化してください。
- 新しいテンプレートを追加する場合は、このディレクトリに WEBP を配置し、`assets/img/articles/index.json` に `key`・`src`・`alt`・`topics` を追記します。
- 既存のファイルはプレースホルダーです。必要に応じて差し替えてください。
